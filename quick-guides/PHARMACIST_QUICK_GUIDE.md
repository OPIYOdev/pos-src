# 💊 PHARMACIST QUICK GUIDE

> **Post at dispensary workstation**

---

## ✅ PRESCRIPTION CHECKLIST

```
✓ Patient name & age
✓ Prescriber license number
✓ Date (<180 days standard / <30 days controlled)
✓ Drug name & strength
✓ Quantity (words AND numbers)
✓ Prescriber signature
```

---

## ⚠️ INTERACTION CHECK

1. Run drug–drug interaction check
2. Check patient allergy profile
3. Verify no duplicate therapy
4. Confirm dose within standard maximum
5. Document all findings in system

---

## 🔐 CONTROLLED SUBSTANCES PROTOCOL

```
TWO-PERSON VERIFICATION required for all transactions
Enter in CS Register IMMEDIATELY after each dispense
Daily count is MANDATORY (06:00 / 09:00 / 17:00 / 18:00)
Report ANY discrepancy within 1 hour to Store Manager
```

---

## 🕐 DISPENSING TIMES (SLA)

| Activity | Standard | Max |
|----------|----------|-----|
| Prescription verification | 3 min | 10 min |
| Drug interaction check | 1 sec | 5 sec |
| Label printing | 5 sec | 30 sec |

---

## 📞 ESCALATION PATH

**Pharmacist → Store Manager → GM → PPB (regulatory)**
