# 🌙 END-OF-DAY CLOSE PROCEDURE

> **Complete in order — do not skip steps**

---

## EOD CHECKLIST

```
[ ] Print X-Z report from POS
[ ] Count cash (2 persons present)
[ ] Compare physical count to system total
[ ] Investigate any variance >KES 200
[ ] Prepare bank deposit slip
[ ] Secure cash in safe
[ ] Sign cashier handover log
[ ] Set system for overnight backup
[ ] Lock pharmacy doors & arm alarm
```

---

## ⏰ EOD TIMELINE

| Time | Activity |
|------|----------|
| 20:00 | Close system / last transaction |
| 20:15 | Count cash by denomination |
| 20:30 | Compare to system total |
| 20:45 | Investigate variance if needed |
| 21:00 | Prepare deposit |
| 21:15 | Secure in safe |

---

## 💵 VARIANCE ACTION QUICK REFERENCE

| Variance | Required Action |
|----------|----------------|
| ≤ KES 200 | Log in system, no further action |
| KES 201–1,000 | Issue written warning, schedule retraining |
| > KES 1,000 | **Formal investigation** — call Store Manager immediately |
| >3 variances / 30 days | Mandatory retraining |

---

## 🗄️ BACKUP CONFIRMATION

Overnight backup runs automatically. Verify green status light on server by 21:30.  
**If backup fails:** SMS IT Support on 0700-123456 immediately.
